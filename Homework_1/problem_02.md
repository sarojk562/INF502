# List Mangler
## Python Program to Transform List Elements

**This program transforms a list by multiplying even numbers by 2 and odd numbers by 3.**

1. The program iterates through each number in the input list.
2. Use the function `list_mangler(list_in)` to transform the list according to the rules.

## Problem

**Write a function with the following signature:** `list_mangler(list_in)`.

```python
# Example usage
input_list = [1, 2, 3, 4, 5]
output_list = list_mangler(input_list)
print(f"Input List: {input_list}")
print(f"Output List: {output_list}")
# Output: [3, 4, 9, 8, 15]
```
