# Calculate Hypotenuse
## Python Program to calculate Hypotenuse given lengths of two sides

**This program returns the length of the hypotenuse of a right triangle using the Pythagorean theorem.**

1. The program uses Python's built-in `math.hypot()` function for accurate calculations.
2. Use the function `pythagoreanTheorem(length_of_a, length_of_b)` to calculate the hypotenuse length.

## Problem

**Write a function with the following signature:** `pythagoreanTheorem(length_of_a, length_of_b)`.

```python
# Example usage
side_a = 3
side_b = 4
hypotenuse = pythagoreanTheorem(side_a, side_b)
print(f"The length of the hypotenuse is: {hypotenuse}")
# Output: 5.0
```
